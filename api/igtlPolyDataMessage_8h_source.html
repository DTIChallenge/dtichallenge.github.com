120  // and <point indecies> are unsigned 32-bit integer.
121  // Consequently, the total size can be calculated by:
122  // sizeof(igtlUint32) * (number of points for 0th cell + 1) + (number of points for 1th cell + 1)

124  // that specifies the number of points in each cell.
