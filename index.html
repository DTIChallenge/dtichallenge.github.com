OpenIGTLink:
Open Network Interface for Image Guided Therapy

OpenIGTLink provides a standardized mechanism for communication among computers and devices in operating rooms for wide variety of image-guided therapy (IGT) applications.

Learn more »

Tracking

For stereotactic surgical guidance, OpenIGTLink provides a simple solution to import 3D tool tracking data from various sensors and commercial surgical navigation system to surgical navigation software, such as 3D Slicer.

View details »

Imaging

OpenIGTLink supports real-time image streaming. It allows you to imoprting intraoperative images to your image processing software from various imaging scanners.

View details »

Control

OpenIGTLink is an ideal solution for integration of robot-assisted intervention systems. It supports sending robot control commands, sensor readings, robot status, etc.
